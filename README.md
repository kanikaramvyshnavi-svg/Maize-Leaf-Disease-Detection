# Maize Leaf Disease Detection using Deep Learning

## Overview
This project uses Deep Learning and Convolutional Neural Networks (CNN) to detect diseases in maize leaves from images.  

The system helps farmers identify plant diseases early and improve crop management.

---

## Features
- Detect maize leaf diseases
- Image classification using CNN
- Image preprocessing
- Model training and testing
- Disease prediction from uploaded images
- Accuracy evaluation

---

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib

---

## Dataset
The dataset contains images of:
- Healthy maize leaves
- Diseased maize leaves

Example disease classes:
- Common Rust
- Blight
- Gray Leaf Spot

---

## Project Workflow
1. Collect dataset
2. Preprocess images
3. Train CNN model
4. Test the model
5. Predict disease from image

---

## Installation

Install required libraries:

```bash
pip install tensorflow keras numpy matplotlib opencv-python
```

---

## Run the Project

```bash
python app.py
```

## Model Architecture
This project uses Convolutional Neural Networks (CNN) for image classification.

CNN helps the model learn:
- Color patterns
- Leaf texture
- Disease spots
- Image features

---

## Output
The model predicts whether the maize leaf is:
- Healthy
- Common Rust
- Blight
- Gray Leaf Spot
